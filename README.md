# 🚀 CODESPRITE - Full-Stack Placement Preparation Arena

An advanced full-stack web application designed to help Computer Science students prepare for technical interviews, analyze their resumes using ATS criteria, and practice coding problems.

## 🌐 Live Deployment Links
* **Frontend (Live Website):** [https://codesprite.netlify.app](https://codesprite.netlify.app)
* **Backend API Gateway:** [https://codesprite-backend.onrender.com](https://codesprite-backend.onrender.com)

## ✨ Features
* **Authentication System:** Secure and professional Signup and Login management.
* **ATS Resume Analyzer:** Parses PDF resumes and extracts match scores along with recommended skill additions.
* **DSA Coding Arena:** Integrates with real-world problem sets tagged by top tech product companies (Google, Amazon, Meta, Microsoft).

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
* **Backend:** Node.js, Express.js, REST APIs
* **Hosting Platforms:** Netlify (Frontend), Render (Backend)
